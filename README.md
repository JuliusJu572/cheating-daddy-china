一个实时 AI 助手,通过屏幕截图与音频分析,在视频通话、面试、演示与会议中提供上下文辅助。

## 功能特性

- **实时 AI 辅助**：支持 AIHubMix API
- **屏幕与音频捕获**：结合屏幕内容与系统/麦克风音频进行上下文分析
- **多档案配置**：内置面试、销售电话、商务会议、演示、谈判等使用档案
- **透明悬浮窗**：窗口始终置顶,可自由移动定位
- **点击穿透模式**：需要时可让窗口对鼠标点击透明
- **跨平台**：支持 macOS 与 Windows；Linux 未支持

## 🪟 Windows 详细安装教程

### 步骤 1：安装应用程序

1. 双击下载的 `.exe` 文件
2. 如果出现"Windows 保护了你的电脑"提示：
   - 点击 **"更多信息"**
   - 然后点击 **"仍要运行"**
3. 按照安装向导完成安装
4. 安装完成后，**先不要打开应用**，继续下面的步骤

### 步骤 2：检查并安装 ffmpeg

#### 2.1 检查是否已安装 ffmpeg

1. **打开命令提示符（CMD）**：
   - 按 `Win + R` 键
   - 输入 `cmd`
   - 按回车键

2. **检查 ffmpeg**：
   - 在黑色窗口中输入：`ffmpeg -version`
   - 按回车键

3. **查看结果**：
   - ✅ 如果显示版本信息（类似 `ffmpeg version 4.x.x`），说明已安装，跳到"步骤 3"
   - ❌ 如果显示 `'ffmpeg' 不是内部或外部命令...`，说明未安装，继续下面步骤

#### 2.2 下载 ffmpeg

1. 访问 [ffmpeg 官网](https://ffmpeg.org/download.html)
2. 点击页面中的 **Windows** 图标
3. 选择 **"Windows builds from gyan.dev"** 链接
4. 下载 **"ffmpeg-release-essentials.zip"**（推荐）或 **"ffmpeg-release-full.zip"**

#### 2.3 安装 ffmpeg

1. **解压文件**：
   - 找到下载的 zip 文件
   - 右键点击，选择 **"全部解压缩"**
   - 解压到 `C:\ffmpeg`（建议路径，方便后续操作）

2. **添加到系统环境变量**：
   - 右键点击 **"此电脑"** 或 **"我的电脑"**
   - 选择 **"属性"**
   - 点击 **"高级系统设置"**
   - 点击 **"环境变量"** 按钮
   - 在 **"系统变量"** 区域找到 **"Path"**，双击打开
   - 点击 **"新建"**
   - 输入：`C:\ffmpeg\bin`（如果你解压到其他位置，相应修改路径）
   - 依次点击 **"确定"** 关闭所有窗口

3. **验证安装**：
   - **关闭之前的 CMD 窗口**（重要！）
   - 重新打开 CMD（按 `Win + R`，输入 `cmd`）
   - 输入：`ffmpeg -version`
   - 应该能看到版本信息了

### 步骤 3：配置应用权限

1. **打开应用程序**
2. 如果出现麦克风/屏幕录制权限请求，点击 **"允许"**
3. Windows 10/11 可能需要在 **设置 → 隐私 → 麦克风/摄像头** 中手动授权

### 步骤 4：开始使用

1. 打开应用程序
2. 在主界面输入加密后的 API Key
3. 在设置中选择使用档案（面试、销售、会议等）
4. 选择语言
5. 点击 **"开始会话"**

## 🍎 macOS 详细安装教程

### 步骤 1：安装应用程序

1. 双击下载的 `.dmg` 文件
2. 将应用图标拖拽到 **"应用程序"** 文件夹
3. **⚠️ 先不要打开应用**，继续下面的重要步骤

### 步骤 2：移除隔离属性（必须操作！）

由于应用未经 Apple 公证，macOS 会阻止运行。需要手动移除隔离属性：

1. **打开终端（Terminal）**：
   - 方法 1：按 `Command + 空格`，输入 `Terminal`，按回车
   - 方法 2：在 **"应用程序"** → **"实用工具"** 中找到 **"终端"**

2. **执行以下命令**（复制粘贴后按回车）：
```bash
   sudo xattr -cr /Applications/Cheating\ Buddy.app
```
   
3. **输入你的 Mac 登录密码**（输入时不会显示，直接输入后按回车）

4. **验证是否成功**：
```bash
   xattr -l /Applications/Cheating\ Buddy.app
```
   - 如果没有任何输出或只有很少输出，说明成功
   - 如果看到 `com.apple.quarantine`，说明还没清除，重新执行步骤 2

### 步骤 3：检查并安装 ffmpeg

#### 3.1 检查是否已安装 ffmpeg

1. **在终端中输入**：
```bash
   ffmpeg -version
```

2. **查看结果**：
   - ✅ 如果显示版本信息，说明已安装，跳到"步骤 4"
   - ❌ 如果显示 `command not found`，说明未安装，继续下面步骤

#### 3.2 下载 ffmpeg

1. 访问 [ffmpeg 官网](https://ffmpeg.org/download.html)
2. 点击 **macOS** 图标
3. 选择 **"Static builds for macOS 64-bit"**
4. 下载最新版本的 `.7z` 或 `.zip` 文件

#### 3.3 安装 ffmpeg（命令行方式）

1. **解压下载的文件**：
   - 双击下载的压缩包自动解压
   - 会生成一个类似 `ffmpeg-6.x.x` 的文件夹

2. **在终端中创建目录**（如果不存在）：
```bash
   sudo mkdir -p /usr/local/bin
```
   - 输入密码后按回车

3. **移动 ffmpeg 到系统路径**：
```bash
   sudo cp ~/Downloads/ffmpeg /usr/local/bin/
```
   - 注意：如果你的 ffmpeg 不在下载文件夹，需要修改路径
   - 比如在桌面，就改成：`sudo cp ~/Desktop/ffmpeg /usr/local/bin/`

4. **赋予执行权限**：
```bash
   sudo chmod +x /usr/local/bin/ffmpeg
```

5. **验证安装**：
```bash
   ffmpeg -version
```
   - 应该能看到版本信息了

### 步骤 4：配置系统权限（重要！）

应用需要以下权限才能正常工作：

#### 4.1 屏幕录制权限

1. 打开 **"系统设置"**（或 **"系统偏好设置"**）
2. 进入 **"隐私与安全性"** → **"屏幕录制"**
3. 点击左下角的 **锁图标** 解锁（输入密码）
4. 找到 **"Cheating Buddy"**，勾选复选框
5. 如果找不到，点击 **"+"** 号手动添加：
   - 进入 **"应用程序"** 文件夹
   - 选择 **"Cheating Buddy"**
   - 点击 **"打开"**

#### 4.2 麦克风权限

1. 在 **"系统设置"** 中
2. 进入 **"隐私与安全性"** → **"麦克风"**
3. 同样找到 **"Cheating Buddy"** 并勾选

#### 4.3 辅助功能权限（如果需要）

1. 进入 **"隐私与安全性"** → **"辅助功能"**
2. 添加 **"Cheating Buddy"** 并勾选

> **⚠️ 重要提示**：设置权限后需要 **完全退出** 应用（在 Dock 上右键点击图标 → 退出），然后重新打开才能生效。

### 步骤 5：首次启动应用

1. **打开 "应用程序" 文件夹**
2. **双击 "Cheating Buddy"**
3. 如果出现 **"无法打开"** 提示：
   - **右键点击** 应用图标
   - 按住 **Option** 键
   - 选择 **"打开"**
   - 在弹出窗口中点击 **"打开"**
4. 输入加密后的 API Key
5. 在设置中选择使用档案和语言
6. 点击 **"开始会话"**

## 🔧 常见问题排查

### Windows 常见问题

**Q: 提示"找不到 ffmpeg"**
- A: 检查环境变量是否配置正确，重启 CMD 或重启电脑

**Q: 应用无法启动**
- A: 右键应用 → 属性 → 兼容性 → 勾选"以管理员身份运行"

**Q: 录制不到系统声音**
- A: 检查系统声音设置，确保立体声混音已启用

### macOS 常见问题

**Q: 提示"已损坏，无法打开"**
- A: 确保已执行 `sudo xattr -cr` 命令移除隔离属性

**Q: ffmpeg 命令找不到**
- A: 检查 `/usr/local/bin` 路径是否在 PATH 中：
```bash
  echo $PATH
```
  如果没有，添加到 `~/.zshrc` 或 `~/.bash_profile`：
```bash
  export PATH="/usr/local/bin:$PATH"
```

**Q: 屏幕录制权限已授予但仍无法截图**
- A: 完全退出应用（Cmd + Q），然后重新打开

**Q: 音频录制失败**
- A: 
  1. 检查麦克风权限
  2. 确保 `SystemAudioDump` 有执行权限
  3. 重启应用
